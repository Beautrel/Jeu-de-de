# 🎲 Jeu de dé — Python/Tkinter

Un jeu simple de simulation de dé avec interface graphique.

## Comment jouer
1. Entrez un nombre entre 1 et 6
2. Cliquez sur **Lancer**
3. Si le dé tombe sur votre chiffre → vous gagnez un point
4. Sinon → vous perdez un point

## Technologies
- Python 3
- Tkinter (interface graphique)
- Pillow (affichage des images)

## Lancer le projet
```bash
pip install pillow
python main.py
```
